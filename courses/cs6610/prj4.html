CS6610 Project 4 - Bump-mapping with Shaders

Project 4 required us to render a scene composed of a room, a bump mapped floor, a cube placed in the centre of the room with a sphere on top of it. We implemented, bump mapping, toon shading with silhouttes and height map to bump map conversion.

Keyboard Controls:

Key Operation
'W/E' Move Light in and out of the screen
'B' Bump Map
'E' Toon SHading
'r' Reset camera position
's' Start/Stop scene rotation

Mouse Controls:

Button Operation
'Left' Rotate the camera
'Right' Pan
'Shift+Right' Zoom in/out

How to run the program?

Run build.sh to compile the code and run.sh to run the program.
