CS6610 Project 3- Shader based Shadow Algorithms

Project 3 required us to render a scene composed of a room, a textured floor, a cube placed in the centre of the room with a teapot on it sized such that it casts shadow on the walls and the floor. We implemented, Projective shadows, Shadow maps and Shadow Volumes in this assignment. For shadow volumes, I have used a triangle occluder as shown in the image.

Keyboard Controls:

Key Operation
'W/E' Move Light in and out of the screen
'P' Projective Shadows
'M' Shadow Mapping
'V' Shadow Volume
'r' Reset camera position
's' Start/Stop scene rotation
'u' Toggle 2D/ MipMap modes
't' Toggle Linear/Nearest filtering for 2D mode
'y' Toggle Linear/Nearest filtering for MipMap mode
'a' Increase/decrease alpha blending on reflective sphere

Mouse Controls:

Button Operation
'Left' Rotate the camera
'Right' Pan
'Shift+Right' Zoom in/out

How to run the program?

Run build.sh to compile the code and run.sh to run the program.
